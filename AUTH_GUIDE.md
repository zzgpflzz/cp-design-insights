# 관리자 인증 가이드

## 단순 패스워드 인증 시스템

Firebase Authentication 없이 간단한 패스워드 방식으로 관리자 기능을 보호합니다.

## 로그인 방법

1. 프론트 페이지(http://localhost:3000) 접속
2. 우측 상단의 **사용자 아이콘(👤)** 클릭
3. 비밀번호 입력: `cpdesign`
4. "확인" 버튼 클릭

## 인증 후 사용 가능한 기능

- **관리자 페이지 버튼** 표시
- 프로젝트 추가/수정/삭제 가능
- 로그아웃 버튼 표시

## 패스워드 변경

패스워드를 변경하려면 `lib/auth.ts` 파일을 수정하세요:

```typescript
const ADMIN_PASSWORD = 'cpdesign'; // 원하는 패스워드로 변경
```

## 보안 참고사항

- 현재 인증 정보는 **세션 스토리지**에 저장됩니다
- 브라우저 탭을 닫으면 자동으로 로그아웃됩니다
- 프로덕션 환경에서는 더 강력한 인증 시스템 사용을 권장합니다

## URL 직접 접근 차단

관리자 페이지(`/admin`)에 직접 접근 시:
- 인증되지 않은 경우 자동으로 메인 페이지로 리다이렉트됩니다
